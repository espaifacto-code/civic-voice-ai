# Demo

This folder contains a simple front-end demonstration of the Participatory Civic AI pipeline.

## Running the demo

From the project root:

```bash
npm install
npm run dev
```

Then visit `http://localhost:5173/` and navigate to **Dashboard**, **Explorer**, or **Process Flow**.

## Notes

- The demo is powered by the React app in the project root under `src/`.
- The main process diagram is available at `/process`.
